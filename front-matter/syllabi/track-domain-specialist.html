FM.3: Course Syllabi

Domain Specialist Track

Prerequisites

Python basics. Domain expertise in healthcare, legal, or finance. Chapters 10 through 11 for LLM API fundamentals. No deep ML knowledge required.

Domain Specialist Track

Applying LLMs to domain problems (medical, legal, business, education) without requiring deep ML expertise.

Learning Sequence

Follow the numbered steps in order. Each step builds on the previous one to give you a coherent understanding of this topic area.

  1. Chapter 00: ML and PyTorch Foundations (skim for vocabulary; skip the code if needed)
  2. Chapter 10: Working with LLM APIs (the practical starting point for using LLMs)
  3. Chapter 11: Prompt Engineering and Advanced Techniques (getting high-quality outputs for your domain)
  4. Chapter 20: Retrieval-Augmented Generation (connecting LLMs to your organization's data)
  5. Chapter 28: LLM Applications (code generation, summarization, search, recommendations)
  6. Chapter 33: LLM Strategy, Product Management and ROI (making the business case, vendor selection, cost planning)
