Course Syllabi: University & Training Programs

How to Use This Section

This section provides four complete 14-week university syllabi and five cross-chapter reading tracks. Instructors can adopt a syllabus directly or mix elements from multiple pathways. All pathways assume a 3-hour weekly session (lecture plus lab). Self-directed learners looking for goal-based paths should see FM.2: Reading Pathways instead.

University Course Syllabi

With 36 chapters, no single semester can cover everything. The following four syllabi are designed for instructors adopting this book for a single-semester (14-week) university course. Click any card for the complete week-by-week syllabus with hyperlinked chapter references.

Cross-Chapter Reading Tracks

If you are reading this book with a specific goal rather than cover to cover, these curated tracks group chapters and sections from across the book into focused learning sequences. Click any card for the full learning sequence with hyperlinked references.

Track 1

Data Engineering Track

Building and curating datasets for LLM training, fine-tuning, and evaluation.
 Track 2

Agent Builder Track

Building AI agents from API calls through multi-agent orchestration, with production deployment.
 Track 3

Researcher Track

Understanding LLM internals, scaling behavior, and frontier research directions.
 Track 4

MLOps / Production Track

Deploying, operating, and maintaining LLM systems in production environments.
 Track 5

Domain Specialist Track

Applying LLMs to domain problems (medical, legal, business, education) without requiring deep ML expertise.
