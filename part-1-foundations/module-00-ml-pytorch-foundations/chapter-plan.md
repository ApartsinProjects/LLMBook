# Chapter Plan: Module 00 - ML & PyTorch Foundations

## Scope
Prerequisite refresher covering core ML concepts and hands-on PyTorch programming.
Ensures all students share a common foundation before diving into NLP and LLMs.

## Target Audience
Software engineers with Python experience but no formal ML background.
Basic linear algebra (vectors, matrices, dot products) assumed.

## Sections (individual HTML files)
1. **section-0.1.html** - ML Basics: Features, Optimization & Generalization
2. **section-0.2.html** - Deep Learning Essentials
3. **section-0.3.html** - PyTorch Tutorial
4. **section-0.4.html** - Reinforcement Learning Foundations

Plus **index.html** as chapter landing page with overview and navigation.

## Learning Objectives
After this module, students will be able to:
- Explain supervised learning, loss functions, and gradient descent
- Describe the bias-variance tradeoff and regularization techniques
- Build and train neural networks from scratch
- Write complete PyTorch training loops with custom datasets
- Explain the RL framework and how it connects to LLM training (RLHF)

## Dependencies
- Prerequisites: Python, basic math
- This module feeds into: Module 1 (NLP foundations), Module 4 (Transformers), Module 16 (RLHF/DPO)

## Navigation Structure
Each section file has:
- Previous/Next section links
- "Up" link to chapter index
- "Up" link to main syllabus

## Style
- Book-like formatting (Georgia serif, justified text)
- Callout boxes: Key Insight (green), Big Picture (purple), Note (blue), Warning (yellow)
- Code examples with syntax highlighting and output
- SVG diagrams for concepts
- Interactive quizzes with expandable answers
- No em dashes or double dashes
