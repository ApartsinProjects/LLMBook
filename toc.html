Building Conversational AI using LLM and Agents

From the mathematics of attention to production agent systems. A practitioner’s guide for engineers, researchers, and students.

About This Book

This book takes you from the mathematical foundations of machine learning through building production AI agent systems. Each chapter combines rigorous explanation with practical code examples, real-world application scenarios, and hands-on exercises. Whether you are an engineer looking to integrate LLMs into your products, a researcher seeking to understand modern architectures in depth, or a student building deep understanding from the ground up, this book provides the knowledge you need at every level of the stack.

What You'll Need

  • Basic proficiency in Python programming
  • Familiarity with Git and the command line
  • Understanding of basic linear algebra and probability (vectors, matrices, distributions)
  • General software engineering concepts (APIs, HTTP, JSON, databases)

What You'll Learn

  • Understand the complete NLP/LLM pipeline from text representations to transformer architectures
  • Use and compare major LLM provider APIs (OpenAI, Anthropic, Google, open-source)
  • Master decoding algorithms, inference optimization, and efficient serving strategies
  • Generate high-quality synthetic datasets for training and evaluation
  • Fine-tune and adapt LLMs using full fine-tuning, LoRA, QLoRA, and other PEFT methods
  • Apply knowledge distillation and model merging to create optimized models
  • Build production RAG systems with vector databases and advanced retrieval strategies
  • Design and implement autonomous AI agents with tool use, planning, reflection, and memory
  • Interpret model internals, design rigorous experiments, and evaluate with statistical rigor
  • Deploy, evaluate, monitor, and secure LLM-based applications in production
