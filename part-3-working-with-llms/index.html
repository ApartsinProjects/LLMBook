Building Conversational AI using LLM and Agents

Part III: Working with LLMs

Practical skills for using LLMs through APIs, prompt engineering, and hybrid ML architectures.

Part Overview

Part III shifts from understanding LLMs to using them effectively. You will learn to work with provider APIs (OpenAI, Anthropic, Google, open-source), master prompt engineering techniques from basic to advanced (chain-of-thought, few-shot, system prompts), and design hybrid systems that combine traditional ML with LLM capabilities for production use cases.

Chapters: 3 (Modules 9 through 11). Builds on the model knowledge from Part II and prepares you for training and adaptation in Part IV.

Module 09 Working with LLM APIs

Hands-on guide to LLM provider APIs: authentication, chat completions, streaming, structured outputs, function calling, cost management, and building robust API clients.

  • 9.1 LLM API Fundamentals and Provider Comparison
  • 9.2 Structured Outputs and Function Calling
  • 9.3 Building Robust API Clients
  • 9.4 Reasoning Models and Multimodal APIs
Module 10 Prompt Engineering and Advanced Techniques

The art and science of getting the best outputs from LLMs: zero-shot, few-shot, chain-of-thought, self-consistency, ReAct, and systematic prompt optimization frameworks.

  • 10.1 Prompt Engineering Foundations
  • 10.2 Advanced Prompting Strategies
  • 10.3 Prompt Optimization and Evaluation
  • 10.4 Production Prompt Management
  • 10.5 Prompting Reasoning and Multimodal Models
Module 11 Hybrid ML+LLM Architectures and Decision Frameworks

When to use LLMs, when to use classical ML, and when to combine both. Covers LLM-as-judge patterns, routing, cascading, cost modeling, and architectural decision frameworks.

  • 11.1 LLM vs. Classical ML Decision Framework
  • 11.2 Hybrid Architecture Patterns
  • 11.3 LLM-as-Judge and Evaluation Patterns
  • 11.4 Cost Modeling and Optimization
  • 11.5 Migration and Integration Strategies
