Multi-Agent Systems

"None of us is as smart as all of us."

Census AI Agent

Chapter Overview

While a single agent equipped with the right tools can accomplish many tasks, the most complex real-world problems benefit from multiple specialized agents working together. Multi-agent systems decompose difficult challenges into subtasks handled by agents with distinct roles, expertise, and capabilities. This mirrors how effective human teams operate: a project manager coordinates, researchers gather information, writers produce content, and reviewers ensure quality.

This module covers the full stack of multi-agent development. It begins with a practical survey of agent frameworks, comparing how LangGraph, CrewAI, AutoGen, and native provider SDKs approach the problem of building agents. It then explores multi-agent architecture patterns including supervisor hierarchies, debate protocols, and pipeline topologies. The module concludes with agentic workflows and pipelines, where you will build production-grade systems with conditional branching, error recovery, checkpointing, and human-in-the-loop oversight.

By the end of this module, you will be able to select the right framework for your use case, design multi-agent architectures that balance autonomy with coordination, and build robust agentic workflows that handle the messy realities of production environments.

Learning Objectives

  • Compare LangGraph, CrewAI, AutoGen, and native SDKs for building agents and multi-agent systems
  • Implement the same agent across multiple frameworks to understand their trade-offs
  • Design supervisor, debate, pipeline, and hierarchical multi-agent architectures
  • Understand conformity effects and groupthink risks in multi-agent deliberation
  • Build state machine workflows with conditional branching, loops, and parallel execution
  • Implement checkpointing, error recovery, and compensation logic for long-running workflows
  • Design human-in-the-loop review gates at critical decision points
  • Build a complete multi-agent research pipeline from planning through review
Fun Fact

Building a multi-agent system is a lot like assembling a heist crew in a movie. You need the planner, the specialist, the smooth talker, and the one who double-checks everything. The only difference is that when your LLM agents argue, they do it in JSON instead of dramatic whispers.

Foundational Papers

Wu, Q., Bansal, G., Zhang, J., Wu, Y., Li, B., Zhu, E., Jiang, L., Zhang, X., Zhang, S., Liu, J., Awadallah, A. H., White, R. W., Burger, D., & Wang, C. (2023). "AutoGen: Enabling Next-Gen LLM Applications via Multi-Agent Conversation." arXiv:2308.08155
Introduces AutoGen's conversable agent framework, enabling flexible multi-agent conversation patterns for complex tasks.
Du, Y., Li, S., Torralba, A., Tenenbaum, J. B., & Mordatch, I. (2023). "Improving Factuality and Reasoning in Language Models through Multiagent Debate." arXiv:2305.14325
Demonstrates that multi-agent debate improves factual accuracy and reasoning beyond single-model baselines.
Hong, S., Zhuge, M., Chen, J., Zheng, X., Cheng, Y., Zhang, C., Wang, J., Wang, Z., Yau, S. K. S., Lin, Z., Zhou, L., Ran, C., Xiao, L., Wu, C., & Schmidhuber, J. (2024). "MetaGPT: Meta Programming for A Multi-Agent Collaborative Framework." arXiv:2308.00352
Proposes a structured multi-agent framework where agents follow software engineering workflows with standardized outputs.
Xiong, K., Ding, X., Cao, Y., Liu, T., & Qin, B. (2025). "Examining LLMs' Deliberative Alignment and Conformity Tendencies." arXiv:2502.10485
Studies sycophantic convergence and groupthink risks in multi-agent deliberation systems at ICLR 2025.
Chan, C., Chen, W., Su, Y., Yu, J., Xue, W., Zhang, S., Fu, J., & Liu, Z. (2024). "ChatEval: Towards Better LLM-based Evaluators through Multi-Agent Debate." arXiv:2308.07201
Applies multi-agent debate to LLM evaluation, showing that adversarial discussion produces more reliable quality judgments.

Key Books

Wooldridge, M. (2009). An Introduction to MultiAgent Systems (2nd ed.). Wiley. cs.ox.ac.uk
The standard textbook on multi-agent systems, covering coordination, negotiation, and organizational structures.
Shoham, Y., & Leyton-Brown, K. (2009). Multiagent Systems: Algorithmic, Game-Theoretic, and Logical Foundations. Cambridge University Press. masfoundations.org
Covers the theoretical foundations of multi-agent interaction including game theory and mechanism design.

Tools & Libraries

LangGraph. LangChain, Inc. github.com/langchain-ai/langgraph
Graph-based framework for building stateful multi-agent workflows with conditional edges and checkpointing.
CrewAI. CrewAI, Inc. github.com/crewAIInc/crewAI
Role-based multi-agent framework that simplifies agent collaboration through crew, task, and agent abstractions.
AutoGen (AG2). Microsoft Research. github.com/microsoft/autogen
Microsoft's multi-agent conversation framework supporting flexible interaction patterns and code execution.
Google Agent Development Kit (ADK). Google. github.com/google/adk-python
Google's open-source toolkit for building multi-agent applications with Gemini models and tool integration.
Smolagents. Hugging Face. github.com/huggingface/smolagents
Lightweight agent library from Hugging Face focused on simplicity and code-based agent actions.
