Appendix U

Docker and Containers for ML/LLM Deployment

Big Picture

Containers provide reproducible, portable environments that eliminate the "it works on my machine" problem that plagues ML projects. This appendix covers Docker fundamentals, Dockerfile authoring for GPU workloads, multi-service orchestration with Docker Compose, containerized LLM inference servers, and production deployment patterns with Kubernetes.
